# 30-Macro-Challenge

Overview
--------

This repository contains a 30-day challenge centered on macroeconomics: data, models, and applied analysis. Each day is a compact project with reproducible code and documentation.

Objectives
---------

- Understand macroeconomic indicators and modeling approaches.
- Apply data analysis and visualization to macro datasets.

Structure
--------

Same standard structure as other challenge repositories: Templates, Days, Docs, Data, Scripts, and Outputs.
